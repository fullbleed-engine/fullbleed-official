# Signature Accessibility Canary

Canary example for text-first signature semantics in `fullbleed.ui.accessibility`.

Exercises:

- `SignatureBlock`, `SignatureStatus`, `SignatureMark`
- decorative vs informative SVG marks
- `Status`, `Alert`, `LiveRegion`
- `FieldSet`, `Legend`, `Label`, `HelpText`, `ErrorText`
- `Details` / `Summary`
- `SrText`
- `A11yContract` validation

Run:

```bash
PYTHONPATH=python python examples/signature_accessibility_canary/report.py
```
